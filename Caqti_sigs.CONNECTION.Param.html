Module Caqti_sigs.CONNECTION.Param

module Param: sig .. end
Parameter encoding functions.
val null : Caqti_sigs.CONNECTION.param
For SQL, null is NULL.
val option : ('a -> Caqti_sigs.CONNECTION.param) ->
       'a option -> Caqti_sigs.CONNECTION.param
option f None is null and option f (Some x) is f x.
val bool : bool -> Caqti_sigs.CONNECTION.param
Constructs a boolean parameter.
val int : int -> Caqti_sigs.CONNECTION.param
Constructs an integer parameter. The remote end may have a different range. For SQL, works with all integer types.
val int64 : int64 -> Caqti_sigs.CONNECTION.param
Constructs an integer parameter. The remote end may have a different range. For SQL, works with all integer types.
val float : float -> Caqti_sigs.CONNECTION.param
Constructs a floating point parameter. The precision of the storage may be different from that of the OCaml float.
val text : string -> Caqti_sigs.CONNECTION.param
Given an UTF-8 encoded text, constructs a textual parameter with backend-specific encoding.
val octets : string -> Caqti_sigs.CONNECTION.param
Constructs a parameter from an arbitrary string. For SQL, the parameter is compatible with the BINARY type.
val date : CalendarLib.Date.t -> Caqti_sigs.CONNECTION.param
Construct a parameter representing a date.
val utc : CalendarLib.Calendar.t -> Caqti_sigs.CONNECTION.param
Construct a parameter representing an UTC time value. Selecting a time zone suitable for an end-user is left to the application.
val other : string -> Caqti_sigs.CONNECTION.param
A backend-specific value.
