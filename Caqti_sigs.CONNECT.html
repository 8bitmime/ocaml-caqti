Module type Caqti_sigs.CONNECT

module type CONNECT = sig .. end
The connect functions as exposed to application code through the concurrency implementations:

  • Caqti_lwt which is provided in the caqti.lwt package if caqti was built with --enable-lwt.
  • Caqti_async which is provided in the caqti.async package if caqti was built with --enable-async.
